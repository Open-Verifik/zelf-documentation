### Endpoint

```
https://api.verifik.co/v2/co/runt/vehicle-by-plate
```

El servicio de Información de Vehículos te permite recuperar datos detallados sobre un vehículo en Colombia proporcionando su placa de matrícula y el número de documento de su propietario. Puedes acceder a información incluyendo la marca, modelo, color, especificaciones técnicas, historial de seguros (SOAT y pólizas de responsabilidad civil) e inspecciones técnicas del vehículo.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parámetros

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentType` | string | Yes | Tipo de documento. Valores permitidos: `CC`, `CE`, `PA`, `RC`, `NIT`. |
| `documentNumber` | string | Yes | Número de documento del propietario del vehículo, sin espacios ni puntos. |
| `plate` | string | Yes | Placa del vehículo a consultar. |

### Notas

- RUNT proporciona información completa del vehículo incluyendo especificaciones técnicas y estado legal.
- SOAT (Seguro Obligatorio de Accidentes de Tránsito) es el seguro obligatorio de accidentes en Colombia.
- El estado "VIGENTE" indica pólizas de seguros activas/actuales.
- El estado del vehículo "ACTIVO" significa que el vehículo está actualmente registrado y operativo.
