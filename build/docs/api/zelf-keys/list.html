List Zelf Keys

Retrieve your stored ZelfKeys records. You can either list records filtered by a specific category, or retrieve a unified list of all your stored data.

Endpoints​

List By Category​

GET https://v3.zelf.world/api/zelf-keys/list?category=:category

List All Categories​

GET https://v3.zelf.world/api/zelf-keys/list-all

Authentication​

Requires a JWT token obtained from POST /api/sessions.

Query Parameters​

For /list Endpoint​

  • category (string, required): The category of records to fetch. Valid options include password, notes, zotp, and credit_card.

Response​

  • 200 OK (By Category)
  • 200 OK (All)
  • 409 Conflict

Examples​

  • Node.js
  • cURL
const axios = require('axios');

// List passwords
const passwordsResponse = await axios.get('https://v3.zelf.world/api/zelf-keys/list?category=password', {
  headers: { 
    'Authorization': `Bearer ${token}`, 
    'Origin': 'https://yourdomain.com' 
  }
});

console.log('Stored passwords:', passwordsResponse.data.data.data);

// List all records
const allRecords = await axios.get('https://v3.zelf.world/api/zelf-keys/list-all', {
  headers: { 
    'Authorization': `Bearer ${token}`, 
    'Origin': 'https://yourdomain.com' 
  }
});

console.log('All Vault Records (Passwords):', allRecords.data.data.data.password);
# List passwords
curl -X GET "https://v3.zelf.world/api/zelf-keys/list?category=password" \
  -H "Origin: https://yourdomain.com" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"

# List all records
curl -X GET "https://v3.zelf.world/api/zelf-keys/list-all" \
  -H "Origin: https://yourdomain.com" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
